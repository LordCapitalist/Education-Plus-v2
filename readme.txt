Dette produkt er skabt i svelte som er et front-end hjemmeside design program som kører på typeskript
Dog for at få programmet til at kører er der et par tiltag der skal tages før at man kan kører programmet

Først skal man havde installeret node.js som er et runtime system der benyttes til udvikling af webapplikationer
Dette er fordi programmet kører på lokal host og derfor ikke er en server men kun fungerer når man aktiverer programmet på egen pc
Node.js kan downloades herfra 
https://nodejs.org/en/

Når programmet er downloaded kan vi bruge dens inbyggede funktion npm install til at få det sidste på plads
Npm install kan installere github kommands/programmer direkte fra konsollen og der er særligt en funktion vi skal bruge.
Funktionen er Npm run som kan bruges til at kører større webapplikationer

Den kan installeres ved denne guide
https://www.npmjs.com/package/npm-run

Eller at man bare skriver 
Npm install -g npm-run 

Når dette er downloaded er man klar til at kører programmet

I din kodeeditor of choice skal man finde vej til hvor filen Education Plus v2 ligger og gå ind i den gennem commandprompt
Her skal du så bruge commandoen 

Npm install 

For at initilize alle filerne så de virker
Derefter skal du så bruge commandoen


npm run dev


Hvor den så ville kører en lokalhostprogram på en lokal server. 
Der ville med dette blive angivet et lokalhost link og når man har trykket på det er man inde i programmet


Her er det meget simpelt sign in med ens email og find loginemailen (den kan være i spam)
Derefter kan man oprette sig og lave sign account, lave en quiz eller prøve en quiz
